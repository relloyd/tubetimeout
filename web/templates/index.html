TubeTimeout Configuration

Key Value What?
Build Time {{.BuildTime}} App build time
Usage Tracker Period {{.UsagePeriod}} The period over which YouTube usage is tracked before it is reset
Next Usage Reset {{.UsageNextReset}} Time the usage tracker will reset
Block Threshold {{.UsageThreshold}} The number of usage minutes to allow this period before blocking kicks in
Current Usage {{.UsageMinutes}} Minutes of usage consumed so far this period
Current Usage Percentage {{.UsagePercent}} Percentage of usage consumed so far this period
Pause {{.PausedUntil}} If paused, the time the usage tracker will resume
